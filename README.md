# AS Crash Course

A simple Android application developed as part of an Android Studio crash course. This project demonstrates basic Android development concepts, including View Binding, handling user input, and updating the UI dynamically.

## 🚀 Features

- **User Input:** Accepts text input from the user via an `EditText`.
- **Interactivity:** A "Submit" button that triggers an action.
- **Dynamic UI Updates:** Displays a personalized greeting message in a `TextView` based on the user's input.
- **View Binding:** Utilizes Android View Binding for safer and more efficient interaction with layout components.

## 🛠️ Built With

- **Kotlin** - Primary programming language.
- **Android SDK** - Target SDK 36.
- **View Binding** - To interact with UI elements.
- **Gradle (Kotlin DSL)** - Build automation system.

## 📦 Getting Started

### Prerequisites

- Android Studio Jellyfish or newer.
- JDK 11 or higher.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yusriamir/ascrashcourse.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Run the app on an emulator or a physical device.

## 📖 Usage

1. Enter your name or any text into the input field.
2. Click the **Submit** button.
3. Observe the greeting message update to "Hello, [your text]".

## 📂 Project Structure

- `app/src/main/java/com/yusriamir/ascrashcourse/MainActivity.kt`: Contains the logic for handling button clicks and updating the UI.
- `app/src/main/res/layout/activity_main.xml`: Defines the user interface layout.
